## Development

### Get the code

```
$ git clone https://github.com/yama07/ProxyDoggoHandler
$ cd ProxyDoggoHandler
$ npm install
```

### Run the app

```
$ npm run dev
```

### Build

```sh
## for macOS
$ npm run build:mac

## for Windows
$ npm run build:win64
### or
$ npm run build:win632
```
